🔵 项目搭建

    创建虚拟环境  virtualenv mezz-env
    激活虚拟环境  source mezz-env/bin/activate

    🔸 安装  pip install mezzanine

    🔸 创建项目

        ⦿ 创建项目并进入     mezzanine-project mezz_dj && cd mezz_dj
        ⦿ 初始化数据库  python manage.py createdb
              域名和端口: 默认  127.0.0.1:8000
              输入超级管理员用户名: v 
              输入邮箱地址密码: xx2610@gmail.com ...  xu...47
              是否安装默认页面: yes
        ⦿ 启动项目 python manage.py runserver
        ⦿ 浏览器 127.0.0.1:8000  进入项目首页
        ⦿ 浏览器 127.0.0.1:8000/admin  进入项目后台
        
        ⦿ 项目就搭建好了.
